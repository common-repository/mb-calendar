MB Calendar - a versatile and powerful calendar/event system for your WordPress website.

== Description ==

**Requirements:**

* PHP 5 or greater
* WordPress 3.0 or greater
* No coding skills

**Features**

* Changeable first day of the week
* Create unlimited calendars
* Each calendar and event can be customized using a wide number of options
* Single and multiple day events
* All day events
* Event priority
* The events can be sorted by date, priority or title
* AJAX front-end

**[UPGRADE NOW](http://codecanyon.net/item/mb-wordpress-calendar/2585635?rel=g0dlik3)** and get more features:

* Custom language support
* Event recursion

**Changelog**

- Fixed a bug where setting the start/end date of an event on the last day of the month to a time higher than 12:00 AM would cause the Events List view to not display that event.

== Installation ==

1. Upload the `mb-wordpress-calendar/` folder to the `/wp-content/plugins/` directory.
2. Activate MB Calendar through the 'Plugins' menu in WordPress.
